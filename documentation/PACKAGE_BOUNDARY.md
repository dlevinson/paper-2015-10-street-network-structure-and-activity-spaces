# Package Boundary

This package is a public-ready supporting-data and documentation package for Parthasarathi, Pavithra; Hochmair, Hartwig; Levinson, David M. (2015). Street network structure and household activity spaces. Urban Studies, 52(6), 1090-1112. https://doi.org/10.1177/0042098014537956.

The paper uses travel survey records from the Twin Cities Travel Behavior Inventory and the 1999 Southeast Florida Travel Survey, downloaded from the Metropolitan Travel Survey Archive, plus Census TIGER/Line street networks, Twin Cities accessibility measures, Florida FSUTMSOnline employment data, and open-source street-network classification software. Those upstream/raw sources are not duplicated here.

The packaged local artifacts are limited to non-person supporting material that was found on the hard drive: a small shopping-accessibility ZIP from the paper folder and a legacy network-measures workbook from the coauthor thesis folder, with modern XLSX/CSV sidecars. Raw household, trip, coordinate, and activity-space microdata are excluded from this package.

The package should be public-safe as staged; it is not a complete executable reproduction bundle. A full reconstruction would require retrieving the public/agency source surveys and rebuilding the activity-space and network measures from the methods described in the paper.
